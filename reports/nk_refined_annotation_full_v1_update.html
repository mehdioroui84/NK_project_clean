0.4 gave enough resolution to separate cytokine-stimulated, tissue-resident, lung, and proliferative structure. We used the clusters as evidence, then collapsed them into a manageable label set.
  • 0.2 was too coarse and merged states we wanted to inspect.
  • 0.3 improved structure but still collapsed several relevant populations.
  • 0.4 increased the cluster count, but the added splits were interpretable and useful for reannotation.

Reannotation Strategy

Practical rule

We kept the label set broad enough for SCANVI training, but preserved splits where clustering, marker programs, and prior annotation agreed.

How the final labels were used

  • B and T were retained as simple reference labels to help SCANVI learn non-NK immune structure.
  • Mature cytotoxic clusters were merged unless a stable marker-supported branch was useful.
  • Cytokine-stimulated cells were split into CCR7+ and proliferative programs.
  • Small non-NK-like populations were kept broad rather than overnamed.
This gives a compact annotation that is still more informative than the original broad labels.

Interpretation

  • B, T, transitional cytotoxic, mature cytotoxic, and lung cytotoxic NK transfer well to held-out datasets.
  • Rare refined states have lower macro-F1 impact because they are small or unevenly represented in held-out data.
  • The gap between weighted and macro F1 is expected for this label set and should be shown transparently.

Where the model struggles

  • Errors concentrate near interfaces between related NK states.
  • The rare refined classes are the main limitation for macro F1.
  • This supports v1 as usable, with a clear list of states to revisit when more held-out evidence accumulates.

How to read this

The refined model is reliable for the dominant biology and reference classes. Rare and dataset-restricted states are useful labels, but we should present them as v1 states with limited zero-shot support.

This is a good model for a first refined reference, not a final biological taxonomy.
